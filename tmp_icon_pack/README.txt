DNS site icon pack

Состав архива:
- favicon.ico
- favicon-16x16.png
- favicon-32x32.png
- favicon-48x48.png
- favicon-64x64.png
- apple-touch-icon.png
- android-chrome-192x192.png
- android-chrome-512x512.png
- mstile-150x150.png
- site.webmanifest
- browserconfig.xml
- html-head-snippet.html
- dns-sim-icon-1024.png

Как установить:
1. Загрузите все файлы в корень сайта.
2. Скопируйте содержимое html-head-snippet.html в <head> вашего сайта.
