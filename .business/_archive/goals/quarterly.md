# Квартальные цели

## Q1 — MVP → рабочая система
Полная карточка, фильтры по МКФ/роли, сравнение с похожими,
модули процессов/самообслуживания/товара, полуавтоматическое обновление,
использование на совещаниях.

## Q2–Q3 — Глубокие модули + роли
Персонал, финансы, локации, клиентский опыт.
5 режимов по ролям. Сравнение с Дивизионом. Node.js API + БД. Python-аналитика.

## Q4 — Автоматизация + промышленная эксплуатация
Автозагрузки, PDF/HTML-экспорт, саммари, уведомления, история решений.
Docker, бэкапы, безопасность. Полный ввод в эксплуатацию.
