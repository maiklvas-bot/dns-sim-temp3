# Этапы внедрения (5 шагов)

1. **ДРРС / аналитический контур** — проверить структуру, логику, красные зоны. Без требований к УМ/ЗУМ.
2. **Пилот: 5–7 филиалов** — разные типы (крупный / ТП / районник / с проблемой / с хорошей управляемостью). Проверить карточку и интерфейс.
3. **Обучение УМ** — короткое совещание (7 шагов): зачем → экран → карточка → красные зоны → отклонения → задачи → эффект.
4. **Обучение ЗУМ (отдельно)** — фокус: операционная дисциплина (поручения, постамат, ТСО, задачи на день).
5. **Регулярное использование на совещаниях** — ежедневно / еженедельно / ежемесячно.

## Главный риск
Система станет «ещё одним отчётом».
**Противоядие:** первый релиз = только: Пульс РРС → карточка → действия → контроль эффекта.
