# Внутренние альтернативы

| Инструмент | Проблема |
|-----------|---------|
| Excel-выгрузки | Несколько версий правды, ручной перенос, ошибки |
| Презентации / защиты | Хороши для защиты, не для ежедневного контроля |
| Внутренние отчёты DNS (1С, FRP, оперативный контроль и т.д.) | Данные есть, но разбросаны — не складываются в управленческую картину |
| Ручные совещания и переписки | Нет владельца, нет срока, эффект не проверяется |

## Позиционирование
Не конкурирует с 1С, FRP, расчётом персонала — берёт из них данные.
> Управленческая надстройка: факт + отклонение + причина + владелец + действие + эффект.
