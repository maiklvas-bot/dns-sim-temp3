# Стоимость владения

## 5 составляющих
1. **Разработка** — проектирование, дизайн, модули, интеграции, тесты
2. **Поддержка данных** — обновление, качество, справочники, методология
3. **Поддержка пользователей** — обучение, инструкции, обратная связь
4. **Инфраструктура** — хостинг, сервер, БД, бэкапы, безопасность
5. **Управленческая поддержка** — владельцы показателей, статусы, эффект

## Варианты инфраструктуры

| Вариант | Когда | Плюсы |
|---------|-------|-------|
| Локальный HTML | MVP | Быстро, просто |
| GitHub + статика | Прототип | Версионирование, Codex |
| Сервер / Docker | Рабочая система | База, роли, история, автообновление |
| Корпоративный BI DNS | При наличии | Безопасность, данные |

## 5 главных рисков

| Риск | Решение |
|------|---------|
| Ручное обновление → ещё один Excel | Автоматизировать частые операции первыми |
| Разные версии данных → споры | Источник + дата + метод + приоритет |
| Дорогая архитектура без пользы | MVP сначала |
| Утечка данных | Только закрытый доступ |
| Нет владельцев показателей | Назначить для каждого модуля |
