# Дорожная карта

## MVP — Месяц 1
Статический HTML + ручная загрузка из Excel/Google Sheets.
Цель: доказать что логика полезна. Не автоматизация — сначала структура.

## Q1 — Рабочая система
Next.js / React. Полная карточка. Фильтры. Сравнение по МКФ.
Модули процессов/самообслуживания/товара. Полуавтоматическое обновление.

## Q2–Q3 — Глубокие модули + роли
Персонал, финансы, локации, клиентский опыт.
Node.js API + PostgreSQL. Python-аналитика. 5 режимов по ролям.

## Q4 — Автоматизация
Автозагрузки, PDF/HTML, саммари, уведомления о красных зонах.
История решений. Docker + бэкапы + безопасность. Полный ввод в эксплуатацию.
