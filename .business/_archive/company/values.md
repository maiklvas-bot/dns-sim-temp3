# Ценности и управленческая философия

## Главный принцип
**РРС Тюмень — система, а не набор магазинов.**

> клиент → локация → филиал → товар → процессы → персонал → мотивация → финансы → действие

## Данные / интуиция / контекст

| Элемент | Роль |
|---------|------|
| Данные | Показывают факт |
| Интуиция/опыт | Гипотеза, не доказательство |
| Контекст | Объясняет почему |
| Действие | Превращает анализ в результат |
| Проверка эффекта | Было ли решение правильным |

## Скорость vs Точность
- **Скорость** — обратимые операционные решения (70–80% данных достаточно)
- **Точность** — дорогие/необратимые (закрытие, CAPEX, штат, мотивация)

## 7 принципов
1. Не верить одной метрике
2. Не делать вывод «сотрудники плохо работают» без проверки системы
3. Клиентский эффект — финальный фильтр
4. Финрезультат не должен ломать будущую модель
5. Филиал без роли в локации — некорректная оценка
6. Продукт оценивается по использованию, не наличию
7. Любой анализ заканчивается действием

## Приоритеты решений
1. Клиентская польза · 2. Финансовая устойчивость · 3. Достоверность данных
4. Роль филиала · 5. Товарная доступность · 6. Процессная скорость
7. Мотивационная корректность · 8. Простота внедрения

## Установка для агентов
> Данные через контекст. Интуиция — гипотеза.
> Финальный критерий: лучше ли клиенту, устойчивее ли экономике, понятнее ли управлению.
