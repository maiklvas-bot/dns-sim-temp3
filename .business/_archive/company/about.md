# О компании

## РРС Тюмень
Региональная Розничная Сеть DNS в Дивизионе Урал.
Уровень: дивизион → **РРС** → филиал/город.
Продукт: цифровая и бытовая техника.

> Всегда «РРС Тюмень» — не «Тюменская область» (в составе есть Ирбит, Свердловская обл.)

## Форматы / роли филиалов

| Роль | Описание |
|------|---------|
| Районник (МКФ 1) | Базовый спрос района |
| Офлайн-концентратор (МКФ 2) | Витрина, поток |
| Супер офлайн (МКФ 3) | Мощный офлайн-магазин |
| Гибрид (МКФ 4) | Офлайн + online + склад |
| Концентратор-гибрид (МКФ 5) | Усиленный гибрид |
| Онлайн-концентратор (МКФ 6) | ИЗ, выдача, набор |
| Суперконцентратор (МКФ 7) | Крупный узел |
| ПВЗ (МКФ 8) | Только выдача |
| РРЦ (МКФ 9) | Склад/распределение |
| Технопоинт (ТП) | Особый формат Тюмени |
| Товарный хаб | Снабжает соседние точки |
| МегаТП | Крупный концентратор КБТ/ТВ |

**ТП1** — Тюмень на Победы ТП
**ТП2** — Тюмень Заречный двор
**ТП3** — Тюмень Окей на Профсоюзной
ТП = Технопоинт (не ТПМ).
