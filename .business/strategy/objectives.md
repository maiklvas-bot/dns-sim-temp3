# Цели

## Главная цель
> **Помогать принимать более правильные управленческие (кадровые) решения.**

SimCenter даёт данные для решения о найме, назначении, развитии и аттестации — но решение остаётся за человеком.

## Цели-опоры
1. **Объективизация оценки** — единая шкала 0–5 и одинаковые кейсы вместо интуиции.
2. **Единая модель компетенций** — общий язык из 14 компетенций для всей РРС.
3. **Скорость и масштаб** — стандартная сессия 20–60 мин вместо дорогого ручного ассессмента.
4. **Развитие людей** — конкретный план по слабым компетенциям, а не только вердикт.

## Чего НЕ делает
- Не выносит автоматическое кадровое решение.
- Не оценивает по бизнес-метрикам магазина (метрики — лишь обратная связь внутри симуляции).
- Не заменяет руководителя — поддерживает его решение данными.

См. [metrics.md](metrics.md) — как измеряем успех, [rollout.md](rollout.md) — масштабирование.
