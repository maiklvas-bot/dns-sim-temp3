# Визуальный стиль

**Источник истины:** `client/src/styles/dns-theme.ts` (дизайн-токены) и `client/public/fonts/` (шрифты). Здесь — смысловая сводка.

## Название
**DNS SimCenter** — центр оценки управленческих компетенций.

## Цвета (из `dns-theme.ts`)
| Токен | Значение | Роль |
|-------|----------|------|
| primary | `#F04E23` | DNS Orange — главный акцент, кнопки, активные элементы |
| primaryLight | `#FF6B35` | светлый оранжевый |
| bgDark | `#0F1923` | тёмный navy — основной фон |
| bgCard | `#1A2634` | фон карточек |
| bgElevated | `#243447` | приподнятые поверхности |
| textPrimary | `#FFFFFF` | основной текст |
| textSecondary | `#94A3B8` | вторичный текст |
| success | `#00C853` | подтверждённый успех |
| warning | `#FFB300` | внимание |
| error | `#FF1744` | риск/ошибка |
| info | `#2979FF` | нейтральная информация |

Тема — **тёмная**. Цвет = статус, не украшение.

## Шрифты
- **Корпоративный:** DNS RotondaC (`client/public/fonts/` — Regular, Bold, Black).
- **Fallback в теме:** Inter / system-ui; моноширинный — JetBrains Mono.

## Статусы
Норма (зелёный) · Внимание (жёлтый) · Риск (оранжевый/красный) · Нет данных (серый).

## Принципы
- Не перегружать цветами; красный/зелёный — только со смыслом.
- Разные экраны для разных ролей.
- Графики результата всегда с управленческим выводом (вердикт, план развития).

> Полные токены (тени, радиусы, градиенты, breakpoints, spacing, z-index) — в `dns-theme.ts`. Не дублировать здесь.
