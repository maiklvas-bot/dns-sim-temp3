# Стейкхолдеры

Кому нужен SimCenter и какой интерес каждый закрывает.

| Стейкхолдер | Интерес | Что получает |
|-------------|---------|--------------|
| **HR / асессоры РРС** | Объективная и быстрая оценка | Инструмент проведения, готовый профиль и отчёт |
| **Директор РРС (ДРРС)** | Качество кадровых решений | Обоснование назначений, единый стандарт по сети |
| **Руководство дивизиона** | Масштабируемость практики | Тиражируемая методология оценки на другие РРС |
| **Кандидаты / руководители** | Понятная обратная связь | Профиль компетенций + план развития |
| **Администраторы системы** | Управляемость контента | Редактирование кейсов, компетенций, медиа, настроек |

## Кто принимает решение по результату
Итоговое кадровое решение остаётся за человеком (ДРРС / HR). SimCenter даёт **данные для решения**, а не выносит его автоматически.

См. [../audience/roles.md](../audience/roles.md) — роли в системе, [../audience/segments.md](../audience/segments.md) — кого оцениваем.
