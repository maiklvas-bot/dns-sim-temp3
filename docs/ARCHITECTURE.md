# Architecture

The project is a TypeScript full-stack application:

- Frontend: React 18, Vite, Tailwind CSS, TanStack Query.
- Backend: Express, TypeScript, SQLite.
- Shared contracts: TypeScript types and schema definitions in `shared/`.
- Runtime: Docker container with persistent bind mounts for SQLite data and uploads.

Main request flow:

1. Browser loads the Vite-built frontend from the Express server.
2. Frontend calls `/api/*` endpoints for auth, content, sessions, admin settings and results.
3. Backend reads/writes SQLite through storage modules.
4. Live simulations are coordinated through backend live-session services and frontend polling/state sync.
5. PDF reports are generated by the backend PDF generator from normalized report payloads.

Security baseline:

- staff login sessions use hardened cookies;
- mutating requests use CSRF-aware client logic;
- passwords are stored as hashes;
- Helmet and rate limiting are enabled;
- secrets are loaded from local `.env*` files, not from Git.
