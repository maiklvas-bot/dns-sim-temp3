# DNS SimCenter — Project Brief

DNS SimCenter is a business simulation for assessing management readiness in a DNS retail-store context.

The product lets an assessor launch a live simulation for a participant, observe progress, manage cases and channel events, review competency results, and export a PDF report.

Primary goals:

- provide a realistic simulation flow for store-management scenarios;
- keep simulation content, scoring and media controlled through the admin area;
- support assessor-led live sessions;
- preserve production data and live-session recovery across deploys;
- prepare changes through GitHub Pull Requests before production releases.

Current source of truth for runtime content remains the existing project content and database. Content must not be replaced from external archives without an explicit task.
