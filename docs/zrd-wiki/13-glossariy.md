# 13. Глоссарий

| Термин | Значение |
|---|---|
| **ЗРД** | Заместитель регионального директора — оцениваемый управленческий грейд |
| **Квартал** | Один ход-цикл партии (4 квартала = год региона); ~12–15 мин |
| **Фаза** | Этап квартала: исследование → действия → событие → производство |
| **Ресурс** | Топливо развития: К/П/Т/С/Р ([03](03-resursy.md)) |
| **К / П / Т / С / Р** | Капитал / Персонал / Технологии / Склады / Потенциал рынка |
| **Запас** | Текущее количество ресурса |
| **Производство (prod)** | Прибавка ресурса/показателя в конце квартала |
| **prodК** | Производство капитала = доход региона (главный движок) |
| **Показатель** | Цель развития: S/N/O, шкала 0–20 ([04](04-pokazateli.md)) |
| **S / N / O** | Продажи / NPS / Охват |
| **prodS / prodN / prodO** | Производство соответствующего показателя |
| **ТР** | Торговый рейтинг = S + N + O + бонус стратегии; итог партии |
| **Порог ранней победы** | S/N/O, достижение всех трёх = досрочная победа |
| **Карта проекта** | Разыгрываемое действие развития ([05](05-karty-proektov.md)) |
| **Стандартное действие** | Всегда доступный дешёвый ход ([06](06-standartnye-deystviya.md)) |
| **Событие** | Кризис/дилемма с вариантами реакции ([07](07-sobytiya.md)) |
| **Базовый урон** | Негативный эффект события, если регион не готов |
| **Множитель наказания** | Коэффициент усиления негатива событий по сложности (×0.5…×2.0) |
| **Декларация цели** | Выбор стратегии в начале партии ([09](09-strategii.md)) |
| **Стратегия** | Сервис / Экспансия / Эффективность |
| **Компетенция (ФАКТ)** | Балл 0–5 по одной из 12 компетенций ЗРД ([10](10-skoring.md)) |
| **§8a** | Принцип неочевидности выбора (анти-хак): оценка по уместности, не по тексту |
| **ctxTags / fitsWhen** | Контекст-теги состояния / теги уместности варианта (механика скоринга) |
| **Намерение (intent)** | Команда хода от клиента/бота, применяемая движком |
| **Seeded RNG** | Детерминированный генератор случайностей (воспроизводимость партий) |
| **Реплей** | Воспроизведение партии из лога для разбора оценщиком |
