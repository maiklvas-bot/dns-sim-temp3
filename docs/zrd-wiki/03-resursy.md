# 03. Ресурсы (5)

Ресурсы — «топливо» развития. У каждого ресурса есть **запас** (сколько сейчас) и
**производство** (сколько прибавляется в конце каждого квартала). Карты и события тратят запас
и/или наращивают производство.

| Ресурс | Код | Что значит | Зачем нужен |
|---|---|---|---|
| **Капитал** | К | Деньги региона | Платит почти за всё; главный ограничитель |
| **Персонал** | П | Кадры | Требуется для магазинов, обучения, выхода в районы |
| **Технологии** | Т | ИТ/оборудование | Гипермаркеты, CRM, BI, e-commerce, автоматизация |
| **Склады** | С | Логистические мощности | Снимают штрафы логистики, открывают ассортимент/гиперы |
| **Потенциал рынка** | Р | Рыночный ресурс | Топливо для рекламных кампаний |

## Ключевая идея: капитал и доход
**Капитал (К)** — связывающий ресурс. Его **производство (prodК) = доход региона** и это
**главный движок партии**. Доход растёт от открытых торговых точек:
- каждый **магазин** даёт **+1 к prodК**,
- **гипермаркет** — **+2 к prodК**,
- **BI-аналитика** — **+1 к prodК**.

Поэтому ранние вложения в торговые точки = более быстрый доход в поздних кварталах. Это
сознательная замена «денежного голода» из ранней версии баланса (см. `docs/zrd-economy-v1.md`).

## Стартовые значения
Зависят от уровня сложности — см. [08. Сложность](08-slozhnost.md). Пример (L3):
запас **К40 / П3 / Т1 / С1 / Р2**, производство **К4 / П1 / Т1 / С0 / Р1**.

## Правила запасов
- Запас не может уйти ниже 0 (нельзя сыграть карту, на которую не хватает).
- Производство ресурса не может стать отрицательным (минимум 0).
- Производство прибавляется к запасу **в фазе производства** (конец квартала).
