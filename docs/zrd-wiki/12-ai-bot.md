# 12. AI-бот

В режиме v1 (solo) игрок соревнуется с **компьютерным соперником**. Бот — **rule-based**
(эвристика по правилам, без «чёрного ящика»): детерминированный, воспроизводимый, объяснимый.

## Как бот принимает решения
- **Декларация цели:** выбирает стратегию по своему «характеру» (стиль).
- **Исследование:** оценивает разложенные карты эвристикой «польза/стоимость» (доход важен;
  карта, двигающая отстающий показатель, ценнее; долгосрочные карты ценнее рано) и оставляет
  лучшие доступные.
- **Действия:** играет самую ценную доступную карту; если нет — стандартное действие на
  отстающий показатель; иначе пас.
- **События:** выбирает вариант, **уместный текущему контексту** (как и ожидается от хорошей игры).

## Уровни/характеры
Сила бота растёт со сложностью ([08. Сложность](08-slozhnost.md)): от пассивного (L1) до
агрессивного+ (L5). Та же система «стилей» используется как **персоны для калибровки скоринга**
(planner / balanced / improviser / weak / risktaker) — см. `docs/zrd-scoring-v1.md`.

| Стиль | Поведение |
|---|---|
| balanced | разумная игра «по ситуации» |
| planner | смотрит данные (BI) перед ходами, строит движок, эффективность |
| improviser | реактивная игра, реже сверяется с контекстом |
| risktaker | агрессивная экспансия |
| weak | слабые выборы (для проверки, что скоринг ловит плохую игру) |

> Источник: `shared/zrd/ai.ts` → `chooseIntent`. Бот — чистая функция состояния (детерминизм).
