# 📘 ЗРД — WIKI симуляции (главное меню)

Полное руководство по бизнес-симуляции **«Институт ЗРД: Покорение новых территорий»** —
как в правилах настольной игры. Объясняет цель, компоненты, ход партии, события и то, как
машина оценивает 12 компетенций. Симуляция вдохновлена *Terraforming Mars*: вы «терраформируете»
торговый регион, поднимая три показателя через сеть магазинов, ресурсы и проекты.

> **Это живой документ.** Он ведётся синхронно с кодом. При любом изменении механики/чисел
> правим и соответствующий раздел Wiki, и `14-changelog.md`. Карта «Wiki ↔ код» — в конце меню.
